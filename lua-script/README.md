# Lua Scripts Collections

A collection of Lua scripts for [YimMenuV2](https://github.com/YimMenu/YimMenuV2).

## Usage

1. Build or download the latest YimMenuV2 release.
2. Launch GTA Online and inject YimMenuV2.
3. Open the **Lua Scripts** tab inside the menu.
4. Place any `.lua` file from this folder into YimMenuV2's `scripts` directory (created on first run, next to the YimMenuV2 executable).
5. Reload scripts from the menu or restart the game.

## Folder Structure

Scripts are organized into the following subfolders:

| Folder           | Description                                                  |
| ---------------- | ------------------------------------------------------------ |
| `unlocks/`       | Content unlock scripts (Arena War, awards, bunker research, collectibles, LSC colors, tattoos, everything, weekly prized car) |
| `businesses/`    | Business management scripts (restock/resupply Hangar, Warehouse, MC businesses, Nightclub, Salvage Yard, Money Fronts) |
| `mission-skips/` | Heist and mission skip scripts (Cayo Perico setup, Cluckin' Bell Raid, Dr. Dre Contract, KnoWay Out, safe cracker) |
| `career/`        | Career progress completion script                            |
| `tools/`         | Utility and gameplay tools (delete personal vehicle, ragdoll, fast respawn, gun van editor, rename editor, snack & armor refiller) |

## Adding a Script

1. Drop your `.lua` file into the appropriate subfolder based on its purpose.
2. Add an entry to the relevant table above with a short description.
3. Keep each script self-contained and document any requirements in a header comment at the top of the file.
