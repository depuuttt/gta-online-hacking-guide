00

Prerequisites

Before you begin
  • An external Stat Editor — or any tool that can write MP stats on GTA V Enhanced.
  • Join an invite-only session before writing any stats. Do not edit while inside an active job or heist.
  • Know your character slot — first character uses $MP0_, second uses $MP1_.
  • Some stats are BOOL (set to 1 / true). Others are INT and need a numeric value — suggested values are given.
01

Character Slot

$MPx_ → $MP0_ / $MP1_

All per-character stats are published as $MPx_…. Replace x with 0 for your first character or 1 for your second. All examples in this guide use $MP0_. $MPPLY_ stats are global (player-wide) and are not slot-prefixed.
