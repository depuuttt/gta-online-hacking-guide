00

Prerequisites

Before you begin
  • GTAHaXUI (external) or another Stat Editor compatible with the Los Santos Tuners update.
  • An Auto Shop must be purchased & set up — you've bought the property and access the contract board inside.
  • Join an invite-only session before writing any stats.
  • Write stats outside an active contract, then start the contract from the Auto Shop board.
01

Character Slot

MP prefix

The Auto Shop stats are published as $MPx_TUNER_…, where x is your character slot. Use $MP0_ for your first character and $MP1_ for your second. All examples in this guide use $MP0_.

02

Select the Active Contract

$MP0_TUNER_CURRENT

The stat $MP0_TUNER_CURRENT sets which Auto Shop contract is currently active on the planning board. Pick the one you want, write the value, then open the board to start it.

$MP0_TUNER_CURRENT

Active Contract

  • 0The Union Depository
  • 1The Superdollar Deal
  • 2The Bank Contract
  • 3The ECU Job
  • 4The Prison Contract
  • 5The Agency Deal
  • 6The LOST Contract
  • 7The Data Contract

Type the stat name in the Name field, enter the value number for the contract you want, and click Write.

03

Skip All Prep Missions

$MP0_TUNER_GEN_BS
$MP0_TUNER_GEN_BS

Skip Preps

Value 12543

Writing 12543 to this stat marks all prep missions for the active contract as complete. After writing this, no preps need to be done — the contract finale is ready to launch from the Auto Shop board.

04

Launch the Finale

From the board

To skip every prep and go straight to the finale, first set $MP0_TUNER_CURRENT to the contract you want, then set $MP0_TUNER_GEN_BS to 12543. Return to the Auto Shop planning board and the finale will be ready to launch — no prep missions required.

Detection & Safety

Use at your own risk

No cheat method is risk-free

Stat editing is considered as safe as any other cheat method, but no approach is completely risk-free. Playing in invite-only sessions with BattlEye disabled reduces risk. As with any stat manipulation, there's always a chance of detection or account action — use at your own discretion.
