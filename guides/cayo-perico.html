00

Prerequisites

Before you begin
  • YimMenu v2 — or an external Stats Editor compatible with GTA V Enhanced.
  • The heist must be set up — you've paid the initial setup cost.
  • You need to complete the Intel mission first, before editing any stats.
  • All stat editing should be done outside the submarine after completing intel.
01

Character Slot

MP prefix

Use $MP0_ for your first character and $MP1_ for your second character. All examples in this guide use $MP0_.

02

Setup & Intel

Step-by-step
01

Pay to set up the heist

Start the heist from the submarine. Pay the initial setup cost to begin the Cayo Perico Heist sequence.

02

Start & complete the Intel mission

Go outside the submarine, teleport to the Velum, and fly to Cayo Perico. Complete the intel mission on the island — you can spawn an Oppressor Mk2 to speed this up.

03

Return to Los Santos

Once intel is complete, return to the city. You're now ready to begin editing stats — remember to stay outside the submarine.

Skip All Prep Missions

$MP0_H4_MISSIONS
$MP0_H4_MISSIONS

Value -1

This marks all prep missions as complete. After writing this, no preps need to be done — the heist finale is ready to launch.

07

Secondary Objectives

Loot control

Secondary loot is controlled by stats that specify what type of loot appears on the Island vs. inside the Compound — plus the value of that loot.

The naming convention uses:

  • I — Island locations
  • C — Compound locations
  • V — Value
Stat Description
$MP0_H4LOOT_CASH_I Cash on the island
$MP0_H4LOOT_CASH_C Cash in the compound
$MP0_H4LOOT_CASH_V Cash value
$MP0_H4LOOT_WEED_I Weed on the island
$MP0_H4LOOT_WEED_C Weed in the compound
$MP0_H4LOOT_WEED_V Weed value
$MP0_H4LOOT_COKE_I Cocaine on the island
$MP0_H4LOOT_COKE_C Cocaine in the compound
$MP0_H4LOOT_COKE_V Cocaine value
$MP0_H4LOOT_GOLD_I Gold on the island
$MP0_H4LOOT_GOLD_C Gold in the compound
$MP0_H4LOOT_GOLD_V Gold value
$MP0_H4LOOT_PAINT Paintings (compound)
$MP0_H4LOOT_PAINT_V Paintings value

You can only set all secondary objectives in a given location to one type. For example, to set all island loot to gold and all compound loot to paintings:

  • $MP0_H4LOOT_GOLD_I = -1
  • $MP0_H4LOOT_PAINT = -1
  • Set all other _I and _C stats to 0

The _V stats let you set the actual monetary value of the loot directly.

09

Detection & Safety

Use at your own risk

No cheat method is risk-free

According to experienced users in the thread, editing heist setup stats is considered as safe as any other cheat method. Playing in invite-only sessions with BattlEye disabled reduces risk. Users report setting finale payouts to maximum values across multiple heists (e.g., $2.5M for Cayo and Bogdan, $3.5M for Diamond Casino) without issues.

However, there's always some chance of detection or account action — so use at your own discretion.

10

GTA V Enhanced

Compatibility
Enhanced

The original YimMenu heist editor does not work with GTA V Enhanced. For the Enhanced version, use an external Stats Editor tool (referenced in the thread as "Stats Editor[External] (Enhanced)"). The stat names and values remain the same.
